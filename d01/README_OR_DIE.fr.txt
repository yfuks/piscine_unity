- Pour lancer la demo du jour, executez le fichier "demoDxx.app" avec
  la commande : "open ./demoDxx.app", ou "Dxx" correspond bien entendu
  au jour present.

- Les commandes pour la demo du jour sont :
  - Gauche et droite pour deplacer les personnages (ou A et D).
  - Espace pour sauter.
  - 1, 2, ou 3 pour selectionner le personnage actif.
  - R pour redemmarer la scene.
  - N pour passer directement au niveau suivant

- Si l'un de vos personnages meurt au dernier niveau l'ecran deviendra
  gris. Il suffit d'appuyer sur R pour restart le niveau.

- Pour importer les assets fournis ce jour, vous devez d'abord ouvrir
  Unity normalement, creer ou ouvrir un projet existant, puis ouvrir
  le fichier "assetsDxx.unitypackage" avec la commande "open
  assetsDxx.unitypackage", ou "Dxx" correspond bien entendu au jour
  present. Une petite pop-up vous demandera confirmation pour
  l'importation des assets. Apres validation, les assets seront
  disponibles dans votre projet.

- Les assets fournis pendant cette piscine sont la propriete de leurs
  auteurs respectifs et ne peuvent etre utilises qu'a des fins
  pedagogiques. Vous ne pouvez donc pas les redistribuer librement.

- Pour le dernier exercice vous n'avez pas d'assets specifiques fournis 
  pour les tourelles et les bullets. A vous de vous debrouiller en tweakant
  certaines images (utilisez la demo comme exemple si vous n'avez pas d'idee)
